# Publish Selected Private Synthesis

## Problem
The local private review blog now contains direction-level technical synthesis. Some of that material is safe and useful for the public blog, but much of the raw source material includes private, sensitive, source-index, or off-topic records. Public publication needs an explicit selection and redaction boundary.

## Goal
Move only public-safe engineering methods from the private synthesis into public Hexo posts. If a theme already has a public article, extend that article. If the theme is not already represented, add a focused public article.

## Non-goals
- Do not publish raw conversation rows, source indexes, local paths, hostnames, ports, or private/admin content.
- Do not disclose project-specific implementation details beyond general open engineering methods.
- Do not modify the private source material directory.
